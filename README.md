Stock Management System (Java)

Description:
This project is a Java-based stock management system developed for a university assignment.
It manages grocery and electronic items, including discounts, taxes, and sorting features.

Features:
- Add Grocery and Electronic items
- Read data from file (data.txt)
- Calculate tax and discount
- Display expired grocery items
- Sort items by brand and price
- Filter items based on discount
- Clone stock list

Technologies Used:
- JAVA
- Object-Oriented Programming (OOP)
- File I/O
- ArrayList
- Comparator

How to Run:
1. Open the project in any Java IDE (e.g., IntelliJ or VS Code)
2. Make sure data.txt is in the correct directory
3. Run the Main class

Notes:
 - Tax for Grocery = 5.75%
 - Tax for Electronics = 16.5%
